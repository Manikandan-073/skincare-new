# MENSKIN — Premium Skincare Landing (Prototype)

This is a lightweight prototype of a premium men's skincare e-commerce front-end.

Files added:
- `index.html` — homepage and UI
- `styles.css` — theme and responsive styles
- `script.js` — client-side interactions (cart counter, quick view, carousel)
- `products.json` — sample product data (now supports an `images` array for each item)
- `script.js` — updated to display the first image and a quick‑view gallery when multiple images exist

How to run:
1. Open `index.html` in a browser (double-click or serve with a simple static server).

Notes & next steps for production:
- Replace placeholder images with optimized product assets.
- Add backend or localStorage persistence for cart and wishlist.
- Implement real authentication and checkout flow.
- Minify CSS/JS and enable image optimization.
- Add unit and E2E tests for interactive features.
